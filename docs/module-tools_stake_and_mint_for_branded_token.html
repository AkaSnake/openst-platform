Module: tools/stake_and_mint/for_branded_token

This script does stake of simple token on the value chain and mint of branded token on the utility chain. Here staker account for ST and beneficiary account for branded token is assumed as member reserve address.

Following are the steps which are performed in this script:
  1. Select the member company for which we want to do stake and mint.
  2. Ask amount to be staked in ST and the passphrase of the member reserve using prompts.
  3. Call stake and mint util for doing the actual transactions on contracts. ref: module:tools/stake_and_mint/util
